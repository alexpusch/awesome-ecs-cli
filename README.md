# Usage
<!-- usage -->
```sh-session
$ npm install -g aecs
$ aecs COMMAND
running command...
$ aecs (-v|--version|version)
aecs/0.0.0 linux-x64 node-v8.10.0
$ aecs --help [COMMAND]
USAGE
  $ aecs COMMAND
...
```
